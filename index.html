Introducción a Bootstrap

¿Qué es bootstrap?

Bootstrap es un framework CSS basado en componentes, una herramienta muy útil para desarrolladores y diseñadores web que buscan mejorar la productividad y la consistencia en el aspecto de sus proyectos.

Ventajas y desventajas:

  • Ventaja: Es muy rápido de usar porque los estilos ya están creados.
  • Desventaja: Tu página puede verse igual a muchas otras si no cambias los colores.
